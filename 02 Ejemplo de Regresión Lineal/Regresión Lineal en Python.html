Recordemos que $$S_{xx} = \sum_{i=1}^{n} (X - \bar{X} )^2 \qquad S_{xy} = \sum_{i=1}^{n} (X - \bar{X} ) \cdot (Y - \bar{Y}) \qquad S_{yy} = \sum_{i=1}^{n} (Y - \bar{Y} )^2$$

Si consideramos el modelo de regresi√≥n lineal simple con intercepto, tenemos los siguientes estimadores para las betas

$$\hat{\beta_1} = \frac{S_{xy}}{S_{xx}} \qquad \hat{\beta_0} = \bar{Y} - \hat{\beta_1} \cdot \bar{X}$$

# Calculamos algunas metricas para ver que tan bueno es el modelo lineal
MAE = np.mean( np.abs(   y     - y_est      )    )
MSE = np.mean( np.power( y     - y_est      , 2) ) # Error Cuadratico Medio     ( MSE = Mean Square Error )
SCT = np.sum(  np.power( y     - np.mean(y) , 2) ) # Suma de cuadrados totales
SCR = np.sum(  np.power( y_est - np.mean(y) , 2) ) # Suma de cuadrados de la regresi√≥n
SCE = np.sum(  np.power( y     - y_est      , 2) ) # Suma de cuadrados del error
R2  = SCR / SCT                                    # R cuadrada

print("MAE: ", MAE)
print("MSE: ", MSE)
